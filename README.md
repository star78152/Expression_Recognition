# 課堂學習狀態影像辨識 <br> Expression Recognition


## 簡介
* 專案於2024年5月-6月  
* 此專案應用場景於電腦教室，聚焦於辨識與分析學生的學習狀態。協助老師在授課過程中掌握學生的注意力，深入了解其學習過程中的狀況與態度。例如，分析學生在操作電腦時的情緒反應、是否專心練習，以及每次專注於電腦前的練習持續時間等資訊，從而提升教學效果。
